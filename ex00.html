DOM e Formulários

DOM

DOM - Data Object Moddel é uma API que visa simplificar a forma de acessar a estrutura de um documento HTML> Facilitando criar, modificar, adicionar, retirar e manipular documentos HTML.

Formulários

São elementos HTML que ppssibilita a interação do usuário com a página web. Como exemplo temos caixas de texto, botões, caixas de seleção, dentre outros. As tags form, label, input são tags de elementos de formulários.

Construa uma interface para que sejam digitados dois números e somados.
